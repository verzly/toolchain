# Contributing

This repository is a public distribution surface for a Verzly toolchain executable. It does not contain the full source code, tests, release workflows, or internal release configuration, so source-code contributions cannot be accepted directly here.

The maintained source lives in `verzly/toolchain`. Use this repository for issues about the published action, README, license, or release assets. Include the tool version, operating system, command, expected behavior, and actual output.

Do not include signing keys, tokens, passwords, keystores, base64-encoded secrets, private release assets, or repository tokens in issues, pull requests, comments, screenshots, or logs.
